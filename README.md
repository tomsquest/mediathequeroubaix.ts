# Typescript CLI for the library of Roubaix (Médiathèque Roubaix)

## Status

**The project is being built** (a couple of minutes per day).

## Usage

Using NPX:

```shell
npx mediathequeroubaix loans renew
```

Using Bun:

```shell
bunx mediathequeroubaix loans renew
```
